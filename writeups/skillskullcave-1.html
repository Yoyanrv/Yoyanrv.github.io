Nmap

Descripción: Nmap (Network Mapper) es una herramienta open-source usada para la exploración de redes y auditorías de seguridad. Se utiliza para descubrir hosts y servicios en una red.

Comandos básicos de uso:


# Escaneo básico de puertos
nmap -p 1-65535 192.168.1.1

# Escaneo de servicios
nmap -sV 192.168.1.1

Casos de uso:

  • Descubrir puertos abiertos en un host.
  • Detección de servicios en puertos específicos.
  • Identificación de versiones de servicios.

# Configurar un proxy en Burp Suite para interceptar tráfico HTTP/HTTPS.

Ghydra

Descripción: Ghydra es una herramienta de ingeniería inversa que permite analizar y descompilar archivos binarios y software compilado para entender su funcionamiento y buscar vulnerabilidades.

Comandos básicos de uso:


# Decompilación de un archivo binario
ghydra -file /path/to/binary

Casos de uso:

  • Reversa binarios para encontrar vulnerabilidades.
  • Descompilar código para análisis estático.
  • Automatización de análisis con scripts personalizados.
